# 何切る問題ビューア (Discard Problem Viewer)

麻雀の「何切る」問題をスマートフォンやPCで快適に閲覧するための、シンプルで軽量なWebアプリケーションです。

## 概要

このアプリは、大量の「何切る」問題画像をスワイプ操作（スマホ）やキーボード操作（PC）でスムーズに切り替えて確認することができます。サーバーレスで動作するため、ローカルファイルをブラウザで開くだけで使用可能です。

## 主な機能

- **スワイプ操作対応**: 左右にスワイプすることで「問題」と「回答」を交互に切り替え、次の問題へ進むことができます。
- **インジケーター表示**: 現在の問題番号（Q1, Q2...）、ページ種別（問題/答え）、および全体の進捗（1/100など）を表示します。
- **PC操作対応**: 左右の矢印キーまたはスペースキーでページをめくることができます。
- **アニメーション**: ページ切り替え時に目に優しいフェード効果を採用しています。
- **画像一括収集・加工**: Webから画像を自動取得するスクリプトと、複数の画像を1枚に結合するユーティリティを同梱しています。

## ファイル構成

```text
discard_problem_app/
├── index.html          # アプリのメイン構造
├── style.css           # デザインとアニメーション
├── viewer.js           # アプリの動作ロジック（スワイプ・表示制御）
├── cards/              # 【重要】問題・回答画像を配置するディレクトリ
├── download_bilibili_images.py # Web記事から画像をダウンロードするスクリプト
├── combine.py          # 小さな画像を縦に結合して1枚の画像にするスクリプト
└── README.md           # このドキュメント
```

## 使い方

### 1. 画像の準備

`cards/` フォルダ内に、以下の規則に従って画像を配置してください。

- 問題画像: `q_001.png`, `q_002.png`, ...
- 回答画像: `a_001.png`, `a_002.png`, ...

> [!TIP]
> 拡張子が `.svg` の場合は、`viewer.js` 内の `CONFIG.imageExtension` を変更してください。

### 2. 設定の変更

`viewer.js` の冒頭にある `CONFIG` オブジェクトを編集することで、表示する問題数などを調節できます。

```javascript
const CONFIG = {
    imageDir: 'cards',      // 画像を保存しているフォルダ名
    totalQuestions: 30,     // 表示したい問題の総数
    imageExtension: 'png',  // 画像ファイルの拡張子 (png, jpg, svg など)
    // ...
};
```

### 3. アプリの起動

1. `index.html` をブラウザ（Chrome, Safari等）で開きます。
2. スマートフォンで閲覧する場合は、フォルダごとスマホに転送し、`index.html` を開けるブラウザアプリ等で使用してください。

## ユーティリティスクリプト

### 画像のダウンロード (`download_bilibili_images.py`)
Bilibiliの記事URLを入力すると、その記事内の画像を自動的に `images/` フォルダへダウンロードします。
※実行には Python が必要です。

### 画像の結合 (`combine.py`)
`Pillow` ライブラリを使用して、複数の画像を縦1枚に結合します。問題と解答が分割されている場合などに便利です。
